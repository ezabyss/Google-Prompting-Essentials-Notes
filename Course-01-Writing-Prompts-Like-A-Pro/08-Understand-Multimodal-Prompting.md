## üß† What Is Multimodal Prompting?

**Multimodal prompting** means using **more than one modality** in a single prompt.

### Modalities = formats such as:

* Text
* Images
* Audio

## üß† Using the Prompting Framework with Multimodal Prompts

You still use:

**Thoughtfully Create Really Excellent Inputs**

But adjust each part slightly:

### Task

* Be specific about **how and why** each modality is used

### Context

* Clarify constraints related to the modalities
* Example: only consider fruits and vegetables in a fridge image

### References

* Can be text, images, or audio
* Make sure the tool supports the references you provide

### Evaluate

* Review carefully across all modalities
* Visual + textual outputs may need extra checking

### Iterate

If output misses the mark:

* Revisit the framework
* Break into smaller tasks
* Add constraints
* Rephrase or use an analogous task
